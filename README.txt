# OpenClassrooms
This repository is just here in order to introduce my workshop on http://exercices.openclassrooms.com
L'objectif de ce fichier README.txt est d'expliquer brièvement mon projet.
L'exercice demandé dans OpenClassrooms est de créer un repository contenant deux fichiers dont chacun aura 4 commits différents.
N'étant pas du tout développeurs je vais partir sur:
- une image (je pense que c'est original comme idée), à laquelle je vais effectuer une série de modifications;
- un fichier texte (je vais partir d'un texte brut, pour montrer différentes étapes en SEO);
